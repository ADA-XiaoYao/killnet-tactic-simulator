## 模拟器功能

本项目将KillNet的核心攻击战术分解为独立的、可交互的Python脚本，并提供了一个混合攻击脚本来模拟其“真实战术形态”。

*   ✅ **流量型攻击 (Volume-based)**
    *   `udp_flood.py`: 模拟无连接的UDP洪水，快速消耗带宽。
    *   `dns_amplify.py`: 模拟利用开放DNS服务器进行反射和放大的攻击。
*   ✅ **协议层耗尽 (Protocol Exhaustion)**
    *   `syn_flood.py`: 模拟TCP半连接攻击，耗尽服务器的连接状态表。
*   ✅ **应用层洪泛 (L7 Flood)**
    *   `http_flood.py`: 模拟高频次的HTTP/S请求，压垮Web应用服务器。
*   ✅ **混合型攻击 (Multi-vector)**
    *   `multi_vector_attack.py`: 可同时编排和启动上述多种攻击向量，模拟真实的复杂攻击场景。

## 系统架构

本模拟器通过 `docker-compose` 一键部署一个完整的攻防实验室，包含以下四个核心组件：

1.  **攻击机 (`attacker`)**: 一个包含所有攻击脚本的容器，我们将在这里执行攻击命令。
2.  **靶机 (`target`)**: 一个简单的Nginx服务器，作为我们所有攻击的目标。
3.  **监控数据采集器 (`prometheus`)**: 负责从靶机收集性能指标。
4.  **可视化仪表盘 (`grafana`)**: 将监控数据以图表形式展示，让我们能实时观察攻击效果。
